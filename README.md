### Q1. 로샤(Rorschach) 검사에서 '형태질'이 가진 임상적 의미를 3가지 쓰시오.

<details>
<summary>🔍 여기를 누르면 정답과 키워드가 보입니다</summary>

*   **필수 채점 키워드:** 현실검증력, 지각의 정확성, 사고의 타당성
*   **모범 답안 예시:**
    1. 현실검증력 파악: 수험자가 현실을 얼마나 왜곡 없이 지각하는지 나타냄
    2. 지각의 정확성: 자극을 일반적인 방식으로 지각하는지 평가
    3. 사고의 타당성: 집단의 공통된 관습적 사고에 동조하는 능력 반영

</details>

### Q2. 행동치료의 기본 가정 3가지를 쓰시오.

<details>
<summary>🔍 여기를 누르면 정답과 키워드가 보입니다</summary>

*   **필수 채점 키워드:** 학습된 행동, 환경적 요인, 행동의 변화
*   **모범 답안 예시:**
    1. 인간의 모든 행동(정상/이상)은 학습된 것이다.
    2. 행동은 환경적 사건에 의해 결정되고 유지된다.
    3. 잘못 학습된 행동은 새로운 학습 원리를 통해 수정될 수 있다.

</details>
